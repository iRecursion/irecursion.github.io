5. 计算数值

  • 总骨重 = 出生年重量 + 出生月重量 + 出生日重量 + 出生时重量；

  • 如：某人在甲子年，正月初一，子时出生；

  • 甲子年：一两二钱，正月：六钱，初一：五钱，子时：一两六；

  • 总骨重 = 1.2 + 0.6 + 0.5 + 1.6 = 3.9；

  • 此处的三两九钱是此人一生的命运；
